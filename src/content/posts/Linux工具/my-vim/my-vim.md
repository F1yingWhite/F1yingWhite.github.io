## vim基本指令
### 基本操作

| 快捷键 |           作用           |
| :----- | :----------------------: |
| ESC    | 从当前模式转换为普通模式 |
| i      |       进入插入模式       |
| :      |      进入命令行模式      |



### 方向键

| 快捷键 | 作用                       |
| :----- | -------------------------- |
| hjkl   | 左下右上                   |
| 0      | 数字0移动到行首            |
| $      | 移动到行末                 |
| ^      | 移动到本行第一个非空格字符 |
| (n)w   | 向前移动n个词              |
| (n)b   | 向后移动n个词              |
| G      | 移动到文件末尾             |
| gg     | 移动到文件开头             |

### 插入

| 快捷键 | 功能               |
| ------ | ------------------ |
| a      | 光标后插入         |
| A      | 行末插入           |
| i      | 光标前插入         |
| o      | 在光标下方插入一行 |
| O      | 在光标上方插入一行 |

### 删除

| 快捷键 | 功能               |
| ------ | ------------------ |
| x      | 删除光标位置的字符 |
| dw     | 删除一个单词       |
| d0     | 删除到行首         |
| d$     | 删除到行末         |
| d）    | 删除到句末         |
| dgg    | 删除到文件开头     |
| dG     | 删除到文件结尾     |
| (n)dd  | 删除n行            |

### 复制与粘贴

| 快捷键   | 功能                              |
| -------- | --------------------------------- |
| yy       | 复制当前行                        |
| p        | 在当前行之后粘贴存储缓冲区的内容  |
| P        | 在当前行前粘贴存储缓冲区的内容    |
| `["x]yy` | 复制当前行至寄存器 x              |
| `["x]p`  | 在当前行之后粘贴寄存器 x 中的内容 |

### 撤销与重做

u:撤销

ctrl+r 重做最后的撤销

### 搜索和替换

| 快捷键 | 功能                 |
| ------ | -------------------- |
| /x     | 在文档的后面部分搜索 |
| ？x    | 在文档的前面部分搜素 |
| n      | 查看前面一个结果     |
| N      | 查看后面一个结果     |
|        |                      |

### 书签

| 快捷键       | 功能                                                         |
| ------------ | ------------------------------------------------------------ |
| `m {a-zA-Z}` | 在当前光标位置设置书签，书签名可用一个大小写字母（{a-zA-Z}） |
| `:marks`     | 列出所有书签                                                 |
| `{a-zA-Z}`   | 跳转到书签 {a-zA-Z}                                          |

### 选择文本

| 快捷键 | 功能             |
| ------ | ---------------- |
| `v`    | 进入逐字可视模式 |
| `V`    | 进入逐行可视模式 |
| `Esc`  | 退出可视模式     |

### 改动选中文本

| 快捷键 | 功能                 |
| ------ | -------------------- |
| `~`    | 切换大小写           |
| `d`    | 删除一个词           |
| `c`    | 变更                 |
| `y`    | 复制                 |
| `>`    | 右移                 |
| `<`    | 左移                 |
| `!`    | 通过外部命令进行过滤 |

### 杂项
(分别更改这些配对标点符号中的文本内容 change inside)
ci’、ci”、ci(、ci[、ci{、ci< -

分别删除这些配对标点符号中的文本内容
di’、di”、di(或dib、di[、di{或diB、di< -

分别复制这些配对标点符号中的文本内容
yi’、yi”、yi(、yi[、yi{、yi< -

分别选中这些配对标点符号中的文本内容
vi’、vi”、vi(、vi[、vi{、vi< -

更改修改的符号,比如"hello world"->'hello world':cs"'(change surround)

想要给当前的单词添加引号,可以使用ysiw"(you surrond inner word)
给整行:yss"

在visual 模式下:使用S"来进行操作
