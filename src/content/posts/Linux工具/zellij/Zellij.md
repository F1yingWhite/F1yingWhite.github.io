基础快捷键如下:
1. 窗口操作：
    - 使用 `ctrl-p + d` 在当前窗口下方新建窗口。
    - 使用 `ctrl-p + r` 在当前窗口右方新建窗口。
    - 使用 `ctrl-p + x` 关闭当前窗口。
2. 使用 `ctrl-q` 退出 zellij 。
